Release Notes

Every release, documented. Download any version or track what's changed.

v1.0.0 Latest March 2025  ·  Initial Release Download
  • New Item Catalogue — manage all maintainable hardware with part numbers, failure rates and categories
  • New LCN Hierarchy — build a full hardware breakdown structure with collapsible tree view
  • New FMECA Register — failure mode and effects analysis with severity grading, MTBF and coverage tracking
  • New Maintenance Tasks — preventive maintenance analysis with intervals, man-hours and skill levels
  • New Support Equipment — catalogue special and standard tools, linked directly to maintenance tasks
  • New Dashboard — live summary with PM burden calculations, FMECA coverage and criticality MTBF cards
  • New S3000L Exports — LSA Tables 010–050 exported as formatted Excel workbooks; full LSAR workbook with cover sheet
  • New Dashboard PDF — one-click PDF summary report of all dashboard metrics
  • New Multi-project support — create, open, rename and close projects; each project is an isolated SQLite database
  • New Custom storage location — choose where new database files are saved
  • New Inline table editing — edit cells directly in place with Tab navigation and auto-save
  • New Column filters — sort and filter on every column in every table
  • New Import from Full LSAR Workbook — reload catalogue, hardware and FMEA data from a previously exported .xlsx

More releases coming soon

Multi-user shared network drive support, login & licensing, and more on the roadmap.
