About

Built for LSA engineers.
Not for everyone.

LSAR Tool is purpose-built software for engineers performing Logistic Support Analysis to the ASD S3000L standard. No bloat, no subscriptions, no internet required.

What is LSAR Tool?

LSAR Tool is a portable desktop application for Windows that helps defence and aerospace engineers manage their Logistic Support Analysis Record (LSAR) data. It covers the full LSA data set — from hardware breakdown and failure mode analysis through to preventive maintenance tasks and support equipment requirements.

Everything runs locally on your machine. Your data never leaves your computer. There are no accounts, no cloud sync, and no telemetry.

ASD S3000L Standard

S3000L is the international standard for Logistic Support Analysis, published by ASD-STAN (Aerospace and Defence Industries Association of Europe — Standardisation). It defines the data model, processes and outputs for LSA on defence and commercial systems.

LSAR Tool structures all data according to S3000L LSA Tables and exports to the standard's tabular format — ready for programme submission or integration with higher-level ILS tools.

LSA Table 010
Hardware Breakdown
LCN hierarchy and item reference data
LSA Table 020
Item Catalogue
Part numbers, failure rates, categories
LSA Table 030
FMECA Register
Failure modes, effects, criticality data
LSA Table 040
Maintenance Tasks
PM tasks, intervals, man-hours, skills
LSA Table 050
Support Equipment
Special and standard tools linked to tasks

Why does this exist?

Most LSA work happens in spreadsheets — fragile, inconsistent, and impossible to query. Enterprise ILS tools exist but are expensive, complex, and built for large programmes with full ILS teams.

LSAR Tool fills the gap: a focused, structured tool that any LSA engineer can download and start using in minutes. It enforces the S3000L data structure while staying simple enough to use solo on a small programme or contract.

Built with

LSAR Tool is an Electron desktop application. All data is stored in a local SQLite database. Exports are generated with ExcelJS. The entire UI is plain HTML, CSS and JavaScript — no front-end framework.

Electron Node.js SQLite (better-sqlite3) ExcelJS HTML / CSS / JS Windows x64
