About Astrelle

Built for ILS engineers.
Not for everyone.

Astrelle is building a suite of purpose-built software tools for Integrated Logistic Support — structured around ASD standards, designed to replace spreadsheets.

What is Astrelle?

Astrelle is a software company building tools for defence and aerospace engineers working in Integrated Logistic Support (ILS). The goal is simple: replace fragile, inconsistent spreadsheet workflows with structured, standard-compliant desktop applications that anyone can download and use.

Every tool in the Astrelle Suite runs fully offline on your machine. Your data never leaves your computer. No accounts, no cloud sync, no telemetry — just the tool and the work.

The Astrelle Suite

ILS is a broad discipline. Astrelle is being built to cover it end to end — one focused tool per domain, all working to the relevant ASD standard.

Why does this exist?

Most ILS work — FMECA analysis, maintenance task planning, technical publication authoring — happens in spreadsheets and Word documents. Tools that do exist are enterprise platforms built for large programme offices with full ILS teams and six-figure budgets.

Astrelle fills the gap. Focused tools that a single engineer or small team can pick up in minutes, on any programme, at any scale.

Built with

Astrelle Suite tools are built as portable Electron desktop applications. All data is stored in local SQLite databases. The entire UI is plain HTML, CSS and JavaScript — no front-end framework, no unnecessary dependencies.

Electron Node.js SQLite (better-sqlite3) ExcelJS HTML / CSS / JS Windows · macOS

License

Astrelle Suite tools are free for personal and non-commercial use. Commercial and programme use (paid contracts, defence programmes, company environments) will require a licence once commercial tiers are available.

The source code is not currently open source.
